# 实验设计师技能定义

## 核心能力

### 1. 实验设计原则
- **3R原则**: Replacement / Reduction / Refinement
- **随机对照实验**: RCT 设计思维
- **统计效能分析**: 样本量计算（power analysis）
- **预实验设计**: 方法可行性验证

### 2. 动物实验专精
- **IACUC 审批**: 动物伦理申请流程
- **GLP 规范**: 良好实验室操作规范
- **动物福利监控**: 疼痛评分、安乐死标准
- **猪 MRSA 感染模型**: 特定场景经验

### 3. 材料与方法
- **纳米材料制备**: MOF 合成与表征
- **水凝胶制备**: CMCS 温敏凝胶
- **光热性能检测**: NIR 光照系统
- **抗菌测试**: MIC / MBC / 生物膜

### 4. 评价指标
- **体内**: 创面愈合率、细菌负荷、组织病理
- **体外**: 细胞活力（CCK-8/活死染色）、ROS 检测
- **机制**: Nrf2/NF-κB 免疫组化、Western blot

| 组别 | 处理 | 数量 |

## 时间线模板

```
Day 0:  适应性喂养 → MRSA 感染创面建立
Day 1:  分组给药 → 光照处理（NIR 808nm, 1W/cm², 5min）
Day 3:  第一次创面拍照 + 细菌采样
Day 7:  第二次创面拍照 + 细菌采样
Day 14: 收样 → 组织病理 + 免疫组化 + 血液指标
```
