# 论文写作专家技能定义

## 核心能力

### 1. 期刊规范
- **Nature 系列**: 简洁、意义导向、150 词摘要
- **Science**: 突破性、新闻性、125 词摘要
- **Cell 系列**: 详细方法、深入讨论、高图表要求
- **生物医学期刊**: ACS / Wiley / Elsevier 各有格式

### 2. 论文章节

#### Title
- ≤120 字符（Nature）
- 包含关键发现
- 避免 "Effect of..." 开头

#### Abstract（结构化摘要）
```
Background: 1-2句背景
Methods: 2-3句方法
Results: 3-4句核心发现
Conclusions: 1-2句意义
```

#### Introduction（引言）
- 三段式：宽背景 → 窄问题 → 研究目的
- 引用最新文献（近 3 年高引）
- 清晰陈述假设

#### Methods（方法）
- 伦理审批号
- 试剂信息（公司 + 货号）
- 设备型号
- 统计方法详细描述

#### Results（结果）
- 按发现逻辑而非实验顺序
- 图表先行，文字补充
- 避免解释和推测

#### Discussion（讨论）
- 先总结主要发现
- 与文献对比（3-5 篇）
- 机制讨论
- 局限性和未来方向

### 3. 语言润色
- **学术词汇表**: 避免口语化
- **动词精准**: "demonstrate" vs "show" vs "indicate"
- **时态规范**: 方法用过去时，讨论用现在时
- **主动 vs 被动**: 现代期刊倾向主动语态

### 4. 图表制作
- **分辨率**: ≥300 dpi（印刷）
- **字体**: Arial / Helvetica，≥6pt
- **配色**: 单色或双栏对比，避免彩虹色
- **图注**: 独立可读，无需正文也能理解

## 常见错误检查清单

- [ ] 首次缩写已定义
- [ ] 数字 1-9 拼写，10+ 用阿拉伯数字
- [ ] 计量单位使用标准缩写
- [ ] 参考文献格式一致
- [ ] P 值写法规范（P = 0.023，*P < 0.05）
- [ ] 伦理声明已包含
- [ ] 利益冲突声明已包含
- [ ] 作者贡献声明已包含
